# supporthub-api (Spring Boot scaffold)

Proyecto base en Spring Boot para implementar desde cero.

## Estructura

Cada modulo mantiene estructura de scaffolding:

- `controller`
- `service`
- `repository`
- `dto`
- `entity`

Modulos incluidos:

- `users`
- `tickets`
- `comments`
- `attachments`
- `notifications`
- `audit-log`

## Ejecutar

```bash
mvn spring-boot:run
```

## Test

```bash
mvn test
```
